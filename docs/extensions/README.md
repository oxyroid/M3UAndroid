# M3UAndroid extensions

[简体中文](README.zh-CN.md)

Choose the guide for your task:

- [Build an extension](developers/README.md): run Hello, implement a typed Hook, and verify the
  result in M3UAndroid.
- [Maintain the extension platform](maintainers/README.md): trace runtime, transport, broker, provider, or result-import failures.

Emby and Jellyfin ship as built-in extensions. External extensions use the same typed Hook
contracts and are currently available as a developer preview.
